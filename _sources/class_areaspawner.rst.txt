AreaSpawner
===========

**Inherits:** :ref:`Area2D<class_Area2D>` **<** :ref:`CollisionObject2D<class_CollisionObject2D>` **<** :ref:`Node2D<class_Node2D>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Spawns Scene as siblings continously in intervals with the option to have multiple spawns in the run. The scene is spawned in a random location inside any of the :ref:`CollisionShape2D<class_CollisionShape2D>` children.



\ **Warning**: If any :ref:`CollisionShape2D<class_CollisionShape2D>` is not a :ref:`RectangleShape2D<class_RectangleShape2D>`, it will be ignored.
