ProjectileAttack
================

**Inherits:** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

A :ref:`Projectile<class_Projectile>` launcher. Suited (but not limited) to long-range or Projectile attacks.

This node is Frame-Synced.

.. rst-class:: classref-property

:ref:`Vector2<class_Vector2>` **PR_Offset** :ref:`🔗<class_ProjectileAttack_property_PR_Offset>`

The offset of the :ref:`Projectile<class_Projectile>`'s center from the parent/shooter's center when the projectile is spawned while the character is facing right.

This will automatically be horizontally flipped if ``PR_Direction`` is ``-1``.

.. _class_ProjectileAttack_property_PR_Direction:

.. rst-class:: classref-property

:ref:`int<class_int>` **PR_Direction** = ``1`` :ref:`🔗<class_ProjectileAttack_property_PR_Direction>`

The direction in which to shoot the projectile relative to the parent.

Intended value is ``1`` or ``-1``. Only the sign of the value will be considered.

.. _class_ProjectileAttack_property_PR_Force:

.. rst-class:: classref-property

:ref:`Vector2<class_Vector2>` **PR_Force** :ref:`🔗<class_ProjectileAttack_property_PR_Force>`

The Force with which to shoot the :ref:`Projectile<class_Projectile>` if the character is facing right.

This will automatically be horizontally flipped by the :ref:`Projectile<class_Projectile>` itself if ``PR_Direction`` is ``-1``.
