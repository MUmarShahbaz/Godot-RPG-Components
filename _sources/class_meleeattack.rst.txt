MeleeAttack
===========

**Inherits:** :ref:`Area2D<class_Area2D>` **<** :ref:`CollisionObject2D<class_CollisionObject2D>` **<** :ref:`Node2D<class_Node2D>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

A Class that can cause damage to any :ref:`Entity<class_Entity>` inside a certain area. Suited (but not limited) to close-range or Melee attacks.

This node is Frame-Synced.

|void| **damage_distributed**\ (\ amount\: :ref:`float<class_float>` = <unknown>\ ) :ref:`🔗<class_MeleeAttack_method_damage_distributed>`
