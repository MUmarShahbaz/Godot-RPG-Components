Entity
======

**Inherits:** :ref:`CharacterBody2D<class_CharacterBody2D>` **<** :ref:`PhysicsBody2D<class_PhysicsBody2D>` **<** :ref:`CollisionObject2D<class_CollisionObject2D>` **<** :ref:`Node2D<class_Node2D>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

**Inherited By:** :ref:`DynamicEntity<class_DynamicEntity>`

Base Class for all 2D Entities. It contains character stats and meta data. 

Handles core logic including:

- *Gravity*\ 

- *Face Direction (and a fixed Container for RayCasts)*\ 

- *Hitpoints*\ 

- *Movement*\ 

- *Animation Control*\ 

- *Basic Audio*\
